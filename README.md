# RavenKit

A modern React starter with Vite, TypeScript, and shadcn/ui.

## Quick Start

```bash
# Create new project
npx create-next-app@latest my-app --template https://github.com/ravvdevv/ravenkit

cd my-app

# Install deps
npm install

# Start dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
